================================================================================
WHERE TO SET DOCKER IMAGE IN RAILWAY (STEP BY STEP)
================================================================================

You're currently in the "Build" section. You need to go to "Source" section!

STEP 1: Go to Source Section
-----------------------------
In Railway Dashboard:
1. Click on your service: "PulmoAI-Assistant_backend"
2. Click on "Settings" tab (top menu)
3. Click on "Source" (left sidebar) ← THIS IS WHERE YOU NEED TO GO!

STEP 2: Change Build Method
---------------------------
In the "Source" section, you'll see:
- Source Repo: Rana-Hassan7272/PulmoAI-Assistant_backend
- Branch: main
- Root Directory: (empty or backend)

Look for "Build Method" or "Builder" option:
- Currently it says: "Railpack" or "Dockerfile"
- Change it to: "Docker Image"

STEP 3: Enter Docker Image
--------------------------
After selecting "Docker Image", you'll see a field:
- Enter: mhassanshahbaz/doctor-assistant-backend:latest
- Click "Update" or "Save"

STEP 4: Do Same for Frontend
-----------------------------
1. Go to Frontend Service → Settings → Source
2. Change to "Docker Image"
3. Enter: mhassanshahbaz/doctor-assistant-frontend:latest
4. Save

================================================================================
IMPORTANT: You're NOT in the right place if you see:
- "Build Command"
- "Custom Build Command"  
- "Metal Build Environment"

You need "Source" section, not "Build" section!
